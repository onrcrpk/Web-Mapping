Terminology

Our Products!

Digital Terrain Model is a topographic model of the bare Earth. It is a mathematical model of the ground surface in which a unique elevation value is assigned to each pixel.

DTM
Digital Terrain Model

Random Forest is a supervised machine learning algorithm widely used for classification. Random Forest consists of a large number of individual decision trees that operate as an ensemble. Random Forest makes prediction based upon the majority vote of the individual predictions made by these decision trees.

RF
Random Forest

Support Vector Machine is a supervised machine learning algorithm. Support Vector Machine finds a hyperplane that separates the data points distinctively and performs classification based on this.

SVM
Super Vector Machine
